# Dystrace
An AI-powered gamified dyslexia early screening tool, based on Bangor Dyslexia Test (BDT).

Overview

Dystrace is a Unity-based application designed to assist with early dyslexia screening. Instead of diagnosing dyslexia, it analyzes children's performance on interactive cognitive tasks and combines the collected data with AI models to estimate the likelihood of dyslexia.

The project aims to provide educators and specialists with a fast, engaging, and accessible screening tool.

Features

🎮 Game-based cognitive assessments

👁️ Eye-tracking support

📝 OCR integration

🎤 Voice command support

🤖 AI-based prediction model

📊 Data collection for research

📱 Cross-platform architecture

Current Status

🚧 The project is currently under active development.

Disclaimer

Dystrace is an early screening tool and does not provide a medical diagnosis. Any results should be interpreted by qualified professionals.
